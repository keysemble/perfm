Perfm
------------

Introduction
------------

A simple filemanager for editing permissions based on PyQT and Python.

Features
--------

-  Bitwise and Octal permissions support

Requirements
------------

-  Python 3.6+

Quickstart
----------

1. Git clone repo
2. Setup virtual env and then pip install -r requirements.txt
2. Download source and run main.py
